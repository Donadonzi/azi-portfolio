OVERVIEW
Chef Koochooloo is a web application that teach STEAM to children grades K - 2, through preparing and cooking a dish from different countries in the world. This version of the web app had put the main focus on the students. Through their pilot program and user feedback, Chef Koochooloo realized, they lacked features which would make it more usable for teachers. I and my other 3 teammates were tasked with coming up with design features to make Chef Koochooloo more functional and usable for teachers using the web app.

Problem

There is no teacher portal or dashboard to allow teachers create an educational plan for the class.

Solution

We designed a portal that teachers can customize contents and modules and search in subjects and contents to increase engaging time.

RESEARCH
Interview: 6 Teachers
Every 10-20 minutes
Teachers try to give students breaks or create transitions to maintain student engagement
Adjustable Lessons
Teachers need lessons that are adjustable for the different learning styles and teaching styles
Common Core
Teachers needed to know what Common Core and/or NGSS standards they are teaching
Student Observation
We observed that the students remained focus at the start of the lesson but as the lesson continued with the same activity, they were noticeably distracted. In both classes, this student disengagement, caused the lesson to be rushed at the end.
TARGET AUDIENCE
Teacher who use Chef Koochooloo application as a secondary educational tool in their classes

Teachers tend to

  • Rearrange content within a lesson
  • Personalize lessons on a class by class basis
  • Plans lessons that are inclusive of different learning styles and promote multisensory learning
  • Create structured lesson plans that include many short activities and physical breaks
  • Use are range of materials (analog and digital) to engage students

Teachers frustrated by

  • Spending more than a few hours preparing a new lesson plan
  • Static content I cannot change
  • Using multiple management tools to track student progress
